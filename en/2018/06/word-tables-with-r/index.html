    Save R Data Frames as Word Tables

    2018-06-15 175 words 1 minute read
    R
    For beginners
    R · Package 'flextable' · Package 'officer' · MS Word

    Summary

    In this blog post, I demonstrate how to save an R data.frame as a Microsoft Word table using R packages flextable and officer. Please, find the R code below.

    R Code

    This is an example of R code to create and save a data.frame as a Word table:

    # A. Load packages
library(tidyverse)
library(flextable)
library(officer)

# B. Import data
data("PlantGrowth", package = "datasets")

# C. Make a summary (i.e., a data frame)
my_summary_as_df <-
    PlantGrowth %>%
    group_by(group) %>%
    summarize("Group size" = n(),
              "Mean weight" = mean(weight),
              "Standard deviation" = sd(weight))

# D. Prepare the data frame for saving
my_summary_to_save <-
    my_summary_as_df %>%
    regulartable()   %>%
    autofit()

# E. Save as a Word table
name_of_word_document <-
    read_docx() %>%
    body_add_flextable(my_summary_to_save) %>%
    print(target = "document_with_summary.docx")

# F. Open the document (Windows only)
shell.exec(name_of_word_document)

# Non-windows users may try this code:
browseURL(name_of_word_document)
